# 蛋白质聚类结果分析报告

# 一、研究目的与应用工具

本报告旨在对AI生成的蛋白质聚类结果进行系统性分析，通过口袋筛选与分子对接实验，挖掘具有潜在功能价值的蛋白质构象及小分子结合特性。本次研究过程中应用的核心工具包括：口袋预测工具fpocket、分子对接工具AutoDocking Vina以及分子动力学模拟工具Gromacs，其中Gromacs主要用于支撑biomenu轨迹聚类文件的生成与处理。

# 二、af-cluster方法聚类结果分析

## 2.1 核心构象筛选与口袋分析

af-cluster方法生成的蛋白质聚类结果中，选取EX_246.pdb、EX_473.pdb、EX_534.pdb作为代表性构象进行后续分析。通过fpocket工具对上述三个构象进行口袋预测与评分后，发现一个高分口袋具有显著的构象特异性——该口袋仅在EX_246.pdb与EX_473.pdb中存在，而在EX_534.pdb中完全缺失。进一步对比两个含口袋构象的口袋评分，EX_473.pdb的口袋分数更高，因此确定其为该特异性口袋对应的最优蛋白质构象，用于后续分子对接实验。

## 2.2 分子对接结果分析

以EX_473.pdb为受体构象，小分子HHS_from_3IP0为配体，使用AutoDocking Vina进行分子对接，共获得10种对接模式，各模式的亲和力及与最优模式的距离参数如下表所示：

|1|-6.4|0.000|0.000|
|2|-6.2|1.732|1.877|
|3|-5.8|2.816|5.473|
|4|-5.8|2.799|5.338|
|5|-5.7|2.263|2.550|
|6|-5.7|18.124|20.251|
|7|-5.6|18.499|20.534|

由对接结果可知，模式1为最优对接模式，其亲和力达到-6.4 kcal/mol，表现出较强的分子间结合能力。模式2的亲和力为-6.2 kcal/mol，与最优模式的RMSD下限仅为1.732，说明该模式下小分子结合构象与最优构象较为接近，结合模式稳定性较好。模式6、7、10的RMSD上下限均超过18 Å，与最优模式的结合构象差异显著，结合亲和力也相对较低，属于非优势结合模式。

## 2.3 构象一致性验证

将EX_473.pdb与polo蛋白晶体构象进行序列比对，结果显示两者的RMSD值为1.540 Å（比对原子数量为975个）。RMSD值较低，表明EX_473.pdb的空间结构与天然polo蛋白晶体构象具有较高的一致性，进一步证明该AI生成构象的合理性与可靠性。

# 三、biomenu轨迹聚类结果分析

## 3.1 构象拆分与口袋筛选

通过biomenu工具对蛋白质分子动力学轨迹进行聚类分析，共拆分得到15个独立构象。采用与af-cluster方法一致的口袋预测流程（fpocket工具），筛选出一个具有构象特异性的口袋——该口袋仅在frame4、frame6、frame7、frame8、frame10、frame12、frame13、frame14这8个构象中存在，在其余7个构象中未检测到。对这8个含口袋构象的口袋评分进行排序，frame8构象的口袋分数最高，确定为该口袋对应的最优受体构象。

## 3.2 分子对接结果分析

以frame8构象为受体，选取与af-cluster方法相同的小分子HHS_from_3IP0为配体，使用AutoDocking Vina执行分子对接，获得10种对接模式，具体参数如下表所示：

|---|---|---|---|

|3|-6.4|2.471|5.481|
|4|-6.4|2.832|4.266|

对比可知，biomenu聚类结果中frame8构象与小分子的结合能力更优：最优对接模式（模式1）的亲和力为-6.6 kcal/mol，低于af-cluster方法中EX_473.pdb的最优亲和力（-6.4 kcal/mol），表明该构象与小分子的结合作用更强。模式2至模式7的亲和力均在-6.0 kcal/mol及以上，且与最优模式的RMSD值相对较小（除模式9、10外），说明存在多种稳定的结合构象，结合模式具有一定的多样性。

## 3.3 构象合理性验证

将frame8构象与holo蛋白构象进行结构比对，测得RMSD值为1.005 Å，该数值显著低于EX_473.pdb与polo晶体构象的RMSD值，表明frame8构象的空间结构与天然holo蛋白的一致性更高，其结构可靠性与功能潜在性更突出。

# 四、核心结论

1. AI生成的蛋白质聚类结果中存在具有构象特异性的高分口袋，且不同聚类方法筛选出的特异性口袋分布特征存在差异：af-cluster方法中高分口袋仅存在于EX_246.pdb与EX_473.pdb中，biomenu轨迹聚类中特异性口袋则分布于8个frame构象中。

2. 小分子HHS_from_3IP0与两种方法筛选出的最优构象均具有较强的结合能力，其中biomenu聚类的frame8构象结合效果更优（最优亲和力-6.6 kcal/mol），且其结构与天然holo蛋白一致性更高（RMSD=1.005 Å），是更具研究价值的蛋白质构象。

3. 两种聚类方法筛选出的AI生成构象均与对应的天然蛋白晶体构象具有较低的RMSD值，证明AI生成蛋白质结构的合理性，为后续的功能研究与药物设计提供了可靠的结构基础。
